# Quiz Game 🎯

A simple text-based quiz game built in Python, played from the terminal (Jupyter Notebook).

## Description
The player answers a series of multiple-choice questions (A, B, or C). The game keeps track of the score and shows the final result at the end.

## How to Play
- Open and run the notebook `Quiz Game.ipynb` from top to bottom.
- Read each question and its three options.
- Type your answer (A, B, or C) and press Enter.
- You earn a point for each correct answer.
- At the end, the game shows your total score.

## Rules
- Single player.
- One point per correct answer.
- Only A, B, or C is accepted (the game asks again for any other input).

## Python Concepts Used
- Lists and dictionaries (questions, options, answers)
- A dictionary for the game state (score, current question, game over)
- Functions (display_welcome, get_valid_choice, etc.)
- Flow control (if / elif / else, while, and for loops)
- Input validation
